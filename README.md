[fpm-ruby](https://github.com/stankevich/fpm-ruby)
======

[fpm-cookery](https://github.com/bernd/fpm-cookery) recipe for building Ruby 1.9.3 .deb package.

## Usage

	fpm-cook
	sudo dpkg -i pkg/*.deb
	fpm-cook clean
