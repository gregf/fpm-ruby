#[fpm-ruby](https://github.com/gregf/fpm-ruby)

[fpm-cookery](https://github.com/bernd/fpm-cookery) recipe for building Ruby 2.1.2 .deb package.

## Usage

	fpm-cook
	sudo dpkg -i pkg/*.deb
	fpm-cook clean
